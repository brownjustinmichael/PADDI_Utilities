:mod:`plots` Module
*******************

.. module:: paddi_utils.plots

This module contains utilities for use when plotting the results of a PADDI run.

Several example plotting scripts are provided, and should be used for guidance on writing scripts to plot with :mod:`paddi_utils`. For example, the following will construct several diagnostic plots about the output::

	plot_diagnostic.py OUT*
